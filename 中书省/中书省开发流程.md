### 第一步：环境准备

在开始之前，请确保你的Ubuntu系统安装了必要的开发库。

1. 打开终端（Ctrl+Alt+T）。

2. 运行以下命令安装依赖：
   
   bash
   
   编辑
   
   ```
   sudo apt update
   sudo apt install python3-gi python3-gi-cairo gir1.2-gtk-4.0 gir1.2-adw-1.0 python3-nautilus python3-magic -y
   ```
   
   *注：`python3-magic`用于检测二进制文件，`nautilus`相关库用于右键菜单。*

### 常见问题与排查

根据你提供的文档，如果遇到问题，可以检查以下几点：

- **右键菜单不显示**：
  - 确认 `~/.local/share/nautilus-python/extensions/` 目录下是否有 `zhongshu-menu.py` 文件。
  - 确认是否重启了 Nautilus (文件管理器)。
- **水波纹效果不生效**：
  - 检查 `main.py` 是否正确加载了 `../data/style.css` 文件。
  - 确保GTK版本是4.0以上。
- **权限拒绝**：
  - 代码中使用了 `pkexec`，如果系统没有配置策略，可能会弹出安全警告。这是正常的，输入密码即可。
